# Description

This repository contains the code associated with the analyses in the study:

**Wu et al., Cross-level Regulatory Interactions Underlying Human Immune Aging**

# Requirements

The following software and R packages are required to run the analyses. Other versions may also work, but the versions listed below were used in this study.

- **R** (v4.1.2)
